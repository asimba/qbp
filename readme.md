# qbp  
  
Simple LZSS+RC32 compression algorithm realization.  
  
--- 
Usage : ./bin/qbp \<OPTIONS\> \<SOURCE\> \<DESTINATION\>  
**Examples:**  
```  
qbp c source_file packed_file  
qbp d packed_file unpacked_file  
```  
---  
**Options**:  
Option  | Value
----- | ------ 
c | compress data  
d | decompress data
